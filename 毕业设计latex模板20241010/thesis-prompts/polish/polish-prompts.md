# 润色与统一风格提示词

## 1. 学术化润色

```text
请对以下论文内容进行学术化润色。

要求：
1. 保持原意不变；
2. 提升语言正式性、逻辑性与连贯性；

4. 使用符合计算机专业本科毕业论文的表达方式；
5. 输出润色后的 LaTeX 正文内容。
```

## 2. 降重改写

```text
请对以下论文内容进行同义改写与结构优化，以降低重复率，但要求：
```

## 3. 统一术语

```text
请统一以下论文内容中的术语表达，要求全文在措辞上保持一致。

统一要求：
1. “仓库管理员”“库存管理员”统一为一个说法；
2. “生产管理员”“生产管理人员”统一；
3. “身份认证”“身份验证”根据上下文统一；
4. “质量追溯”“质检追溯”根据语境规范化；
5. “前后端分离架构”“B/S 架构”“系统总体架构”等概念避免混用。

请直接输出统一后的正文内容，并保持学术化风格。
```
