# 第6章 系统测试

```text
请撰写毕业论文第6章“系统测试”，对象为一个基于 Spring Boot、Vue 3、MySQL 的企业供应链协同管理系统。

请按以下结构展开：
6.1 测试环境与工具
6.2 单元测试
6.3 功能测试
6.4 测试结果分析

【写作要求】
1. 测试环境可包括操作系统、JDK、Maven、Node.js、MySQL、浏览器等；
2. 单元测试应围绕认证模块、订单流程、生产流程、采购流程、质检流程、周计划生成服务等展开；
3. 功能测试应覆盖登录认证、订单处理、库存管理、生产流转、采购协同、供应商处理、质量追溯、实时消息通知等；
4. 测试结果分析需体现系统功能基本满足设计要求，同时客观指出局限性；
5. 采用学术论文写法，不写成用户操作说明；
6. 输出 LaTeX 格式。
```
